# 番茄钟计时器 — 规格说明

> 版本: 1.1 | 日期: 2026-07-02 | 状态: APPROVED (Eng Review 修复后)

## 项目概述

一个番茄钟计时器：25 分钟工作 + 5 分钟休息，循环计时，有开始/暂停/重置功能。

## 技术栈概览

- **框架**: React + TypeScript
- **构建工具**: Vite
- **CSS**: Tailwind CSS
- **测试**: Vitest + React Testing Library + @testing-library/jest-dom + jsdom
- **运行方式**: 纯本地，`npm install && npm run dev`

## 设计约束

| 约束项 | 决定 | 原因 |
|--------|------|------|
| 持久化 | 不需要 | 用户确认不需要 localStorage 或历史记录 |
| 音效/通知 | 不需要 | 用户确认不需要倒计时结束提醒 |
| 自定义时长 | 硬编码 | 25分钟工作 / 5分钟休息，不需要可配置 |
| 部署 | 不需要 | 纯本地 `npm run dev` 使用 |

## 用户故事

> 作为一个需要集中注意力工作的人，我想使用番茄钟来管理我的工作和休息时间，提高效率。
