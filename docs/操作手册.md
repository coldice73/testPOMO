# 番茄钟计时器 — 操作手册

> 版本: 1.0 | 日期: 2026-07-02

---

## 快速开始

### 环境要求

- Node.js >= 18
- npm >= 9

### 安装与运行

```bash
# 1. 进入项目目录
cd pomodoro

# 2. 安装依赖
npm install

# 3. 启动开发服务器
npm run dev
```

浏览器会自动打开 `http://localhost:5173`，即可开始使用。

## 使用指南

### 界面说明

```
┌──────────────────────────────┐
│                              │
│        🍅 工作时间            │  ← 模式标签
│                              │
│          ╭─────╮             │
│         │  25:00 │           │  ← 进度环 + 倒计时
│          ╰─────╯             │
│                              │
│    [ 开始 ]  [ 重置 ]         │  ← 控制按钮
│                              │
└──────────────────────────────┘
```

### 基本操作

| 操作 | 按钮 | 说明 |
|------|------|------|
| 开始计时 | 点击"开始" | 倒计时从当前时间开始每秒递减 |
| 暂停计时 | 点击"暂停" | 倒计时暂停，再次点击"开始"继续 |
| 重置计时 | 点击"重置" | 回到当前模式的初始时间，停止计时 |

### 工作流程

```
25:00 工作 → 00:00 → 自动切换 → 05:00 休息 → 00:00 → 自动切换 → 25:00 工作 → ...
```

1. 打开页面，初始状态为 **🍅 工作时间 25:00**
2. 点击"开始"，专注工作 25 分钟
3. 倒计时归零后，自动切换到 **☕ 休息时间 05:00**
4. 休息 5 分钟后，自动切换回工作时间
5. 如此循环

### 视觉提示

- **红色进度环**: 当前处于工作时间
- **蓝色进度环**: 当前处于休息时间
- **进度环填充**: 表示已消耗的时间比例（环越长 = 剩余时间越少）
- **浏览器 Tab 标题**: 实时显示 `[MM:SS] 🍅/☕ 模式 - 番茄钟`，切到其他窗口也能看到

## 常见问题

### Q: 倒计时不准怎么办？
A: 本应用使用 `Date.now()` 时间差计算，而非 `setInterval` 固定间隔，能有效避免浏览器后台 tab 节流导致的计时漂移。

### Q: 可以自定义工作时长吗？
A: 当前版本固定为 25 分钟工作 + 5 分钟休息。如需自定义，请在代码中修改 `WORK_DURATION` 和 `BREAK_DURATION` 常量（位于 `src/hooks/useTimer.ts`）。

### Q: 倒计时结束有提醒吗？
A: 当前版本仅通过页面视觉变化（模式标签切换、进度环颜色变化）和浏览器 Tab 标题变化来提示，没有声音或弹窗通知。

### Q: 关闭浏览器后进度会保存吗？
A: 不会。当前版本不包含持久化功能，每次打开都是全新的 25:00。
