# 番茄钟计时器 — 项目PRD

---

## 项目概述

一个番茄钟计时器：25 分钟工作 + 5 分钟休息，循环计时，有开始/暂停/重置功能。

---

## 功能需求

1. 显示当前模式：🍅 工作时间 / ☕ 休息时间
2. 显示倒计时：MM:SS 格式，大字
3. 开始按钮：开始倒计时
4. 暂停按钮：暂停倒计时，再次点开始继续
5. 重置按钮：重置为当前模式的初始时间（25:00 或 05:00）
6. 工作时间 25:00 倒计时到 00:00 → 自动切换到 05:00 休息
7. 休息时间 05:00 倒计时到 00:00 → 自动切换到 25:00 工作（循环）
8. 页面标题实时更新：浏览器 tab 上显示剩余时间 + 模式
9. 进度环：SVG 圆形进度条，显示当前模式已消耗比例

---

## 技术栈

- **框架**: React + TypeScript
- **构建工具**: Vite
- **CSS**: Tailwind CSS（原子化样式，App.css 仅用于 Tailwind 无法覆盖的全局样式）
- **测试**: Vitest + React Testing Library + @testing-library/jest-dom + jsdom
- **运行方式**: 纯本地，`npm install && npm run dev`

## 自定义 Hook：useTimer

```typescript
src/hooks/useTimer.ts

参数: 无
返回: { timeLeft, mode, totalTime, isRunning, progress, start, pause, reset }

内部逻辑:

状态:
  mode: 'work' | 'break'  初始 'work'
  timeLeft: number         初始 WORK_DURATION（1500）
  isRunning: boolean       初始 false
  lastTick: number | null  上次 tick 的时间戳 (Date.now())

常量:
  WORK_DURATION = 25 * 60  // 1500 秒
  BREAK_DURATION = 5 * 60  // 300 秒

useEffect:
  if (!isRunning) return
  
  lastTick = Date.now()
  const id = setInterval(() => {
    const now = Date.now()
    const elapsed = Math.floor((now - lastTick) / 1000)
    if (elapsed >= 1) {
      lastTick = now  // 重置基准时间戳
      setTimeLeft(prev => {
        if (prev <= 1) {
          // 先返回 0，显示 00:00 一帧
          if (prev === 1) return 0
          // 已经是 0 了 → 切换到新模式，保持运行
          switchMode()
          return currentModeDuration()
        }
        return prev - elapsed  // 扣除实际经过的秒数（Date.now() 时间差）
      })
    }
  }, 200)  // 高频检查 (200ms)，保证精度
  return () => clearInterval(id)  // cleanup: 组件卸载时清理

start():
  if (isRunning) return  // 幂等守卫: 防止快速双击
  setRunning(true)
  lastTick = Date.now()

pause(): setRunning(false)

reset():
  setTimeLeft(currentModeDuration())
  setRunning(false)

switchMode():
  setMode(prev => prev === 'work' ? 'break' : 'work')
  // 注意: switchMode 不改变 isRunning — 自动切换后保持运行

currentModeDuration():
  return mode === 'work' ? WORK_DURATION : BREAK_DURATION

进度计算 (派生值，非独立状态):
  progress = 1 - (timeLeft / totalTime)
  totalTime = currentModeDuration()
```

## 组件设计

### TimerDisplay.tsx

```
Props: timeLeft (number), mode (TimerMode), progress (number)
渲染:
  - 模式标签: "🍅 工作时间" / "☕ 休息时间"
  - 倒计时: MM:SS 格式（大字居中）
  - ProgressRing: SVG 进度环环绕倒计时
```

### ControlButtons.tsx

```
Props: isRunning (boolean), onStart, onPause, onReset
渲染:
  - isRunning → 显示"暂停"按钮 / 否则显示"开始"按钮
  - 始终显示"重置"按钮
  - 按钮样式区别明显
```

### ProgressRing.tsx

```
Props: progress (number 0-1), size (number), strokeWidth (number)
渲染:
  - SVG circle
  - 背景环（灰色）
  - 前景弧（工作=红色/休息=蓝色），stroke-dasharray + stroke-dashoffset 实现
  - 从顶部开始，顺时针
```

```
SVG 进度环实现：
<svg width={size} height={size}>
  <circle
    cx={center} cy={center} r={radius}
    fill="none"
    stroke="#e2e8f0"
    strokeWidth={strokeWidth}
  />
  <circle
    cx={center} cy={center} r={radius}
    fill="none"
    stroke={color}
    strokeWidth={strokeWidth}
    strokeDasharray={circumference}
    strokeDashoffset={circumference * (1 - progress)}
    strokeLinecap="round"
    transform={`rotate(-90 ${center} ${center})`}
  />
</svg>
```

## 倒计时格式化

```typescript
function formatTime(totalSeconds: number): string {
  const minutes = Math.floor(totalSeconds / 60)
  const seconds = totalSeconds % 60
  return `${String(minutes).padStart(2, '0')}:${String(seconds).padStart(2, '0')}`
}
```

## 验收标准

1. 初始状态显示 25:00，模式为"🍅 工作时间"
2. 点开始 → 倒计时每秒减少 1
3. 点暂停 → 倒计时停止，点开始 → 继续
4. 25:00 → 00:00 → 自动切换到 05:00，"☕ 休息时间"
5. 05:00 → 00:00 → 自动切换到 25:00，"🍅 工作时间"
6. 重置 → 回到当前模式的初始时间，停止计时
7. 浏览器 tab 标题实时更新
8. 进度环随倒计时减少而缩短
